# Blinkist-to-Markdown bookmarklet

A simple bookmarklet to convert a Blinkist book summary to Markdown. 

![Screenshot of the bookmarklet in action](screenshot.png)

## How to use

1. Copy the code in bookmarklet.js, create a new bookmark and paste the code into the address field.
2. Log into your Blinkist account and find the book you want to read.
3. Click the bookmarklet and a modal will open containing the converted text.

NOTE: If you're using Safari you'll need to enable 'Allow JavaScript from Smart Search field' in the developer settings.
